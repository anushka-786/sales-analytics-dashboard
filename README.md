# Sales Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard built using the Superstore dataset. It provides insights into sales performance, profit trends, regional sales, and product category analysis through interactive visualizations.

## Dashboard Features
- Sales Performance Analysis
- Profit Analysis
- Regional Sales Comparison
- Monthly Sales Trends
- Category & Sub-Category Analysis
- Interactive Filters and Slicers
- KPI Cards

## Tools Used
- Power BI
- Microsoft Excel
- Power Query

## Dataset
Sample Superstore Dataset

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- Business Intelligence
- Dashboard Design
- KPI Analysis
